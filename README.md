# DynamicForm
Use Json or User Interface  to generate form



## Back End with Visual Studio 2017 up

### setup
```
Update-database -v  (/data)
```

### Development 
```
Set the web project as start project
Run
```

### Excute 
```
Dotnet Run (/web)
```






## Front End Only

### setup
```
npm install
```

### Development
```
npm run serve
```

### Production
```
npm run build
```
